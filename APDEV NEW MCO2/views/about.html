Lab Reservation

About the System

The Lab Reservation System is a web-based platform that helps students and laboratory technicians reserve computer lab slots, manage bookings, and monitor availability in a faster and more organized way.

Core Features

  • User authentication through login and registration
  • Lab slot reservation with seat and time selection
  • Walk-in reservation support for technicians
  • User profile viewing and editing
  • Reservation tracking, updating, and management
  • Real-time monitoring of available lab slots

Technology Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js with Express
  • Database: MongoDB

NPM Packages & Third-Party Libraries Used

  • express Backend web framework used for routing, middleware handling, and server-side request processing.
  • mongoose MongoDB object modeling library used to define schemas and handle database operations.
  • express-session Handles user session management for authentication and login persistence.
  • connect-mongo Stores session data in MongoDB so login sessions can persist properly.
  • bcrypt Used for password hashing to improve account security.
  • nodemon Development dependency that automatically restarts the server whenever code changes are made.
    Development Only

Project Information

This system was developed as part of an academic project to improve laboratory scheduling, reduce manual coordination, and provide a more convenient reservation experience for both students and laboratory technicians.
